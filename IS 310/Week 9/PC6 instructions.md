**Write a bash script that does the following tasks:**

1.  Ask the user for their first and last name and concatenate them into a full name.
    
    *   Print the full name.
        
2.  Ask the user for their city and state.
    
    *   Print out the city and state.
        
3.  Ask the user for their degree and major.
    
    *   Format and print the degree and major as: "Degree: \[degree\], Major: \[major\]"_._
        
4.  Ask the user for their 3 favorite tech companies.
    
    *   Store the companies in an array and print them one by one with numbering.
        
5.  Ask the user for 5 exam scores.
    
    *   Calculate and print the average score and the lowest score.
        
6.  Ask the user for a sentence describing their major.
    
    *   Save the description to a file named major\_description.txt.
        

_**Please submit the .sh file with the name as: (**_**PC\_6.sh**_**)**_

**Grading Rubric:**

*   Deduct 5 points if program fails to seek the inputs as stated above.
    
*   Deduct 5 points if the program does not run.
    
*   0 if code is copied from the internet or someone else's program.
